Synchronize

This page demonstrates a way to produce a stereo image across two applets for use in GeoWall with twin, polarized projections. The two applets are jmolAppletA and jmolAppletB. Once the checkbox is checked, the image in the applet on the right is generated from the other applet. All mouse manipulations should be done with the applet on the left. You can set the degrees of rotation by sending the command set stereoDegrees n to the controlling applet (on the left) where n is an angle. The default is -5 degrees rotation.
